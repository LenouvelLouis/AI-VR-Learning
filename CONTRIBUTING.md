# Contributing to Museum Quiz VR

Thank you for your interest in contributing to Museum Quiz VR!

## Getting Started

1. Fork the repository
2. Clone your fork locally
3. Create a new branch for your feature or fix
4. Make your changes
5. Test thoroughly on Meta Quest
6. Submit a pull request

## Development Setup

### Prerequisites

- Unity 2022.3 LTS or later
- Meta Quest SDK
- Visual Studio or VS Code with C# support

### Code Style

- Follow C# naming conventions
- Use meaningful variable and method names
- Keep methods focused and concise
- Add XML documentation for public APIs

### Commit Messages

Use clear, descriptive commit messages:

```
feat: Add new monument to the museum
fix: Correct quiz scoring calculation
docs: Update installation instructions
refactor: Simplify API response handling
```

## Pull Request Process

1. Ensure your code compiles without errors
2. Test on Meta Quest hardware if possible
3. Update documentation if needed
4. Fill out the PR template completely
5. Request review from maintainers

## Reporting Issues

When reporting issues, please include:

- Unity version
- Meta Quest model
- Steps to reproduce
- Expected vs actual behavior
- Screenshots or logs if applicable

## Questions?

Feel free to open an issue for questions or discussions.
